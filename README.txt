DzayerMall - النسخة الكاملة (Backend) - Unhackable Edition
========================================================

هذه النسخة تحتوي على تطبيق ويب متكامل باستخدام Python (Flask) مع أعلى معايير الأمان والحماية.

المتطلبات:
- Python 3.x
- pip

طريقة التشغيل:
1. قم بفتح سطر الأوامر (Terminal) في هذا المجلد.
2. قم بتثبيت المكتبات اللازمة:
   pip install -r requirements.txt

3. قم تشغيل التطبيق:
   python app.py

4. افتح المتصفح على الرابط:
   http://127.0.0.1:5000

المميزات الأمنية المتقدمة (Advanced Security Features):
-------------------------------------------------------
1. المصادقة الثنائية (2FA):
   - دعم كامل لتطبيقات المصادقة (Google Authenticator, Authy).
   - عند تسجيل الدخول لأول مرة، سيطلب منك مسح رمز QR لتفعيل الحماية.

2. مصيدة الهكر (Honeypot):
   - حقل مخفي في نموذج تسجيل الدخول لا يراه البشر.
   - إذا قام "بوت" بملء هذا الحقل، يتم حظر المحاولة فوراً.

3. حماية ضد هجمات CSRF:
   - تم تفعيل `CSRFProtect` لمنع تنفيذ أوامر غير مصرح بها من مواقع أخرى.

4. ترويسات أمنية صارمة (Strict Headers):
   - استخدام `Flask-Talisman` لفرض سياسة أمان المحتوى (CSP) ومنع Clickjacking و XSS.

5. تحديد معدل الطلبات (Rate Limiting):
   - حماية ضد هجمات التخمين (Brute Force) بحد أقصى 5 محاولات في الدقيقة لتسجيل الدخول.

6. ملفات تعريف الارتباط الآمنة (Secure Cookies):
   - تفعيل `HttpOnly` و `SameSite=Strict` لمنع سرقة الجلسات (Session Hijacking).

7. التحقق الصارم من المدخلات (Input Validation):
   - استخدام `WTForms` لضمان أن البيانات المدخلة مطابقة للمعايير وتطهيرها من أي أكواد خبيثة.

ملفات المشروع:
- .env: ملف الإعدادات السرية.
- app.py: الخادم الرئيسي.
- models.py: قاعدة البيانات.
- forms.py: نماذج الإدخال الآمنة.
- templates/: صفحات HTML (بما في ذلك صفحات 2FA الجديدة).
